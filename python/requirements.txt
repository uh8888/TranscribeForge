# Hinweise zur Plattform-Kompatibilität:
# - macOS Catalina (10.15): System-Python 3.8 ist zu alt. Python 3.10 via
#   pyenv installieren (`pyenv install 3.10.13`). torch 2.x liefert keine
#   offiziellen Catalina-Wheels mehr — falls Installation fehlschlägt,
#   torch auf 1.13.1 pinnen und pyannote.audio auf 2.1.1 zurücksetzen
#   (Comment-Block unten). Wer auf macOS 11+ ist, nutzt die Defaults.
# - macOS 11+ / Linux: Defaults funktionieren out-of-the-box.
# - GPU: CPU-Variante ist hier gepinnt. Für CUDA via `pip install torch
#   --index-url https://download.pytorch.org/whl/cu121` nachziehen.
